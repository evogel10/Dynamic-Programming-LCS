The source code in this assignment can be compiled from the command line in the Terminal.

To compile the program, locate the directory where the files are in and enter: javac EVogelLab3.java Stopwatch.java

To run the program, locate the directory where the complied files are in and enter: java EVogelLab3 Input.txt Output.txt

The program will read in the Input.txt file and write to the Output.txt file the valid sequences paired up and compared to findy the longest common subsequence as well as associated staticstics and invlaid inputs.
